Code Map — which file calls which layer

wifi_manager.c Layer 2

Manages the WiFi connection to the router and syncs the clock via NTP. Other services call wifi_manager_connected() to check if internet is available before making requests.

→ Layer 3b — WiFi HAL (talks to C6 via esp-hosted)
esp_wifi_init() esp_wifi_connect() esp_wifi_start() esp_wifi_set_config() esp_sntp_init()
→ Layer 3a — FreeRTOS
xEventGroupCreate() xEventGroupWaitBits() xEventGroupSetBits()
weather_service.c Layer 2

Fetches weather and AQI from open-meteo.com every 10 min. Waits for WiFi and NTP to be ready first, then updates the home screen widgets directly.

→ Layer 3b — LWIP + mbedTLS (internet)
esp_http_client_init() esp_http_client_perform() cJSON_Parse()
→ Layer 3a — FreeRTOS
xTaskCreate(weather_task) vTaskDelay()
→ Layer 1 — updates LVGL widgets directly
lvgl_acquire() lv_label_set_text() lv_arc_set_value() lvgl_release()
reads from Layer 2 — other services
wifi_manager_connected() wifi_manager_time_synced() fota_active (flag)
modbus_inverter.c Layer 2

Polls the inverter every 3 s via RS485. Builds a Modbus RTU request frame, sends it, reads the response, decodes the registers, and writes the decoded values into the global gd struct that the screen reads from.

→ Layer 3b — UART + GPIO hardware drivers
uart_write_bytes() uart_read_bytes() uart_driver_install() gpio_set_level(DE_PIN) gpio_config()
→ Layer 3a — FreeRTOS
xTaskCreate(modbus_task) vTaskDelay()
→ Layer 1 — writes decoded data for the screen
lvgl_acquire() gd.solar_kw = ... gd.batt_v = ... gd.load_kw = ... lvgl_release()
fota.c Layer 2

Checks GitHub for a newer firmware version, downloads the binary if one exists, writes it to the inactive flash partition, and reboots. Sets fota_active=true so weather_service pauses during the download.

→ Layer 3b — LWIP + mbedTLS (download)
esp_http_client_perform() esp_crt_bundle_attach()
→ Layer 3b — OTA flash ops
esp_ota_begin() esp_ota_write() esp_ota_end() esp_ota_set_boot_partition()
→ Layer 3a — FreeRTOS
xTaskCreate(fota_task) vTaskDelay()
reads from Layer 2 — other services
wifi_manager_connected()
